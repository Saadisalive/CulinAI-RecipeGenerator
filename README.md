## 📖 About

**CulinAI** is an AI-powered recipe generator that helps you discover creative and delicious recipes in seconds. Simply enter the ingredients you have—or describe the meal you want—and let AI generate a complete recipe with cooking instructions.

This project was built as a test project using **Bolt** to explore rapid AI application development.

## 💡 How It Works

1. Open the application.
2. Enter your ingredients or describe the meal you want.
3. Click **Generate**.
4. Let AI create a complete recipe.
5. Enjoy cooking!
